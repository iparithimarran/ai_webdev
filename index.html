About this project
What is this?

A handwritten digit recognizer powered by a Convolutional Neural Network trained on the MNIST dataset. Draw any digit from 0 to 9 — however messy — and the model predicts what it is in real time.

How it works

Your drawing is exported as a base64 PNG, sent to the Flask backend via POST /predict, resized to 28×28 pixels, passed through the CNN, and the 10-class softmax output is returned as confidence scores.
